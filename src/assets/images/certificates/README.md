# Certificate Images

Letakkan gambar sertifikat Anda di folder ini dengan nama file:

1. `telkom-cyber-security.jpg` - Sertifikat Bootcamp Telkom Digiup Cyber Security
2. `oracle-database.jpg` - Sertifikat Oracle Database
3. `umk-itfest.jpg` - Sertifikat UMK ITFEST Web Development Competition

**Format yang disarankan:**
- Format: JPG, PNG, atau WebP
- Ukuran: Landscape (16:9 atau 4:3)
- Resolusi: Minimal 800x450px untuk hasil terbaik
- File size: Maksimal 500KB per gambar (compressed)

**Tips:**
- Pastikan gambar cukup jelas dan readable
- Jika sertifikat terlalu panjang, crop bagian yang penting (nama, judul, tanggal)
- Gunakan tools seperti TinyPNG untuk kompres gambar tanpa mengurangi kualitas
