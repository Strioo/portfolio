## Image Guidelines

- **Format**: JPG or PNG
- **Size**: Keep under 500KB for optimal loading
- **Quality**: High resolution but optimized for web
- **Aspect Ratio**: 16:9 (1200x675px) or 4:3 (1200x900px) recommended
- **Content**: Show the main interface/dashboard of each project

## Note

If images are not uploaded, the cards will display a placeholder with the project name.
